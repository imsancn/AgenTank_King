# 技能：Stun（扰乱）

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能名称 | stun |
| 调用方式 | `me.stun()` |
| 持续时间 | 6 帧 |
| 冷却时间 | 25 帧 |
| 消耗 | 无额外消耗，仅需冷却完毕 |

## 技能效果

激活后扰乱敌方坦克的操控系统，使其在 6 帧内转向和移动指令被随机化。敌方输入的 `turn("left")` 或 `turn("right")` 可能变成相反方向，`go()` 可能变成无效动作或其他方向移动。

## 使用条件

使用前必须检查以下条件：

1. `me.skill` 存在且 `me.skill.type === "stun"`
2. `me.skill.remainingCooldownFrames === 0`（冷却已结束）
3. 建议在敌方正在移动或需要精确操作时施放

## 代码示例

```javascript
function onIdle(me, enemy, game) {
  // 敌方在视野内且 stun 可用时施放
  if (me.skill &&
      me.skill.type === "stun" &&
      me.skill.remainingCooldownFrames === 0 &&
      enemy.tank) {
    me.stun();
    return;
  }

  // 正常战斗逻辑
  if (enemy.tank) {
    me.fire();
  } else {
    me.turn("right");
  }
}
```

## 使用策略

- **干扰瞄准**：在敌方试图瞄准你时施放，使其转向混乱，无法有效射击
- **破坏走位**：敌方在复杂地形中移动时施放，可能导致其撞墙或走进死路
- **持续时间优势**：6 帧的持续时间较长，可在这段时间内安全输出
- **星点争夺**：在敌方冲向星星时施放，使其偏离路线
- **冷却优势**：25 帧冷却较短，使用频率高于 freeze 和 cloak

## 注意事项

- stun 不会完全阻止敌方行动，只是随机化操控，敌方仍有可能做出正确动作
- 持续 6 帧但冷却仅 25 帧，覆盖率较高，适合频繁施压
- 对静止不动的敌方坦克效果有限，因为不需要转向和移动
- stun 与 freeze 的区别：freeze 完全停止行动 2 帧，stun 扰乱操控 6 帧
- 最佳使用场景是敌方正在复杂走位或精准瞄准时
