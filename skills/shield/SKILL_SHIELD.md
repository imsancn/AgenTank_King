# 技能：Shield（护盾）

## 基本信息

| 属性 | 值 |
|------|-----|
| 技能名称 | shield |
| 调用方式 | `me.shield()` |
| 持续时间 | 最多 4 帧 |
| 冷却时间 | 25 帧 |
| 消耗 | 无额外消耗，仅需冷却完毕 |

## 技能效果

激活后为坦克生成一层护盾，护盾存在期间可阻挡敌方子弹。护盾在以下情况会提前消失：

- **被击破**：累计挡下 2 发子弹后护盾破碎
- **超时**：超过 4 帧后自动消散

## 使用条件

使用前必须检查以下条件：

1. `me.skill` 存在且 `me.skill.type === "shield"`
2. `me.skill.remainingCooldownFrames === 0`（冷却已结束）
3. 建议在敌方有子弹射来时激活，最大化护盾价值

## 代码示例

```javascript
function onIdle(me, enemy, game) {
  // 检测到敌方子弹且护盾可用时激活
  if (me.skill &&
      me.skill.type === "shield" &&
      me.skill.remainingCooldownFrames === 0 &&
      enemy.bullet) {
    me.shield();
    return;
  }

  // 正常战斗逻辑
  if (enemy.tank) {
    me.fire();
  } else {
    me.go();
  }
}
```

## 使用策略

- **防弹时机**：最佳使用时机是预判敌方即将射击或已射击时，用护盾吸收关键子弹
- **节奏控制**：25 帧冷却意味着大约每 25 帧可使用一次，需合理规划使用频率
- **星点争夺**：在争夺星星时激活护盾，可在敌方火力压制下安全拾取
- **子弹上限**：护盾最多挡 2 发，面对多发子弹（如 overload 双发）时需谨慎

## 注意事项

- 护盾不能阻挡 freeze、stun、poison 等控制类技能
- 护盾被击破后立即进入冷却，不会等满 4 帧
- 如果激活时没有敌方子弹，护盾会在 4 帧后白白浪费
- 优先用于保命而非进攻，shield 的核心价值是防御
